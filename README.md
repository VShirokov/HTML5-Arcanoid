# HTML5-Arcanoid
HTML5 Canvas + JS

You can play on the demo page: https://vshirokov.github.io/HTML5-Arkanoid/

It is a simple arkanoid using native Javascript and HTML5 canvas. In the future I will add features:
1) Menu Interface
3) Multiple levels
